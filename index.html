Circular Progress Bar

An easy-to-use HTML web component.

Easy-to-use

With CPB, displaying a circular progressbar is just one line of code.

Animated

CPB has a CSS3 animation system built-in.

Border support

Easily add a border to the progress bar.

Color customization

CPB uses HTML color to choose filling, background or border color.

Really animated

All properties can easily be animated.

Customizable

If all this is still not enough, CPB allows an in-depth customization through CSS.
